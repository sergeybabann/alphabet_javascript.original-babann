---------- 1. Переменные и константы в JavaScript ---------- 

Переменная - это именованная ссылка на область памяти, которая используется для хранения, чтения, 
записи (изменения) информации.

Константа - это именованная ссылка на область памяти доступная только для храненияи чтения информации.
1. Идентификатор (именование ссылки) не может быть переназначен
2. В некоторых случаях значение может быть изменено, например, если константа указывает на объект, то 
свойства объекта могут быть изменены.

Особенности использования констант:
1. Константа должна быть сразу инициализирована, а не только объявлена.
2. Константа не может быть переопределена.

Объявление (declared) переменной означает ее поименование без присвоения начального значения. 
Полностью к работе она еще не готова, хранит значение undefined

Инициализация (initialization) переменной означает, что:
    1. переменная объявлена
    2. ей присвоено начальное значение 


Объявление переменной:

let name // объявление переменной

Инициализация переменной:

let name = 'Alex' // инициализация переменной

Переопределение переменной происходит без ключевого слова let:

name = 'Olga' // переопределение переменной 

Вывод переменной в консоль:
Используя метод console.log далее передаем имя переменной в скобки. 
let age // объявление переменной, она не инициализирована (ей не было присвоено значение), поэтому она имеет 
значение undefined
console.log(age) // вывод переменной в консоль

Объявление и инициализация 2-ух переменных в одной строке:
let a = 0, b = 0

Присвоение 2-ум переменным одного строковго значения:
let a = "A"
let b = a // кладем значение переменной a в переменную b  
Аналогичный результат в одну строку:
let a,b = a = "A"
